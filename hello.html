Os comandos principas para connector a um reparatorio GITHUB :

  1. git clone "link do reparatorio"
  2. touch nomedoficheiro.html
  3. git add nomedoficheiro
  4. git commit -m "observação do que se fez"
  5. git status : nos permite ver onde nos situamos no repatorio
  6. git push : publicar as alterções
  7. git commit -am "observação da alteração que se fez"
